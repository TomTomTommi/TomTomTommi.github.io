SouthKen SV: A Real-world Stereo Video Dataset

TPAMI 2026

Junpeng Jing, Ye Mao, Anlan Qiu, Krystian Mikolajczyk
Imperial College London
Paper IEEE Code Datasets

Abstract

Current stereo video datasets are limited in both diversity and realism. There is no diversified real-world stereo video dataset for visualization and qualitative comparison. To compensate for the data shortage, we release a real-world dataset SouthKensington Stereo Video Dataset (Southken SV).
